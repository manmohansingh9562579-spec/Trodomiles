# Trodomiles 🌍

Trodomiles is a modern travel discovery website built using Next.js, TypeScript, and Framer Motion.

Users can explore beautiful destinations, discover famous places, and enjoy a smooth animated UI experience.

##  Live Website
https://trodomiles.vercel.app

##  Tech Stack
- Next.js
- TypeScript
- Framer Motion
- Tailwind CSS

## ✨ Features
- Modern UI
- Animated transitions
- Destination pages
- Responsive design
- Fast performance

## 📦 Run Locally

```bash
npm install
npm run dev
```
